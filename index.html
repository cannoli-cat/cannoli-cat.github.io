Weft, An In-Game Scripting Language

View on GitHub

A sandboxed scripting language and bytecode VM for Unity. Scripts compile to bytecode and run on a gas-limited stack machine with closures, cooperative multitasking, and a modular runtime that lets you toggle language features per-project.

C# Unity Bytecode VM Compiler Design
Why
Let players and designers write real game logic at runtime without exposing a full language or risking infinite loops.
What
A complete pipeline from lexer to parser to compiler to stack-based VM, with closures, upvalue tracking, process scheduling, and an attribute based binding system for exposing C# to scripts.
Impact
Enables emergent player solutions and replayability through scripting.
